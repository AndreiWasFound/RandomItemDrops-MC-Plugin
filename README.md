# RandomItemDrops MC Plugin
 This plugin changes the item it drops when you break an specific item. This plugin comes with an config.yml, so you can config what block drops what. For example:
 
 blocks:
   DIRT:
     - DIAMOND
	 - IRON_INGOT
	 - GOLD_BLOCK
	 - COAL_BLOCK
	 - STONE
	 - FLINT
	 
and so on...
